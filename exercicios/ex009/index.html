Trabalhando com lista

Listas Ordenadas

Ultilizando a tag <ol></ol> "Ordered List"

Para adicionar itens a uma lista é só adicionar a tag <li> item </li> "List Item"

  1. Acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes

Listas não Ordenadas

Ultilizando a tag <ul></ul> "Unordered List"

Para adicionar itens a uma lista é só adicionar a tag <li> item </li> "List Item"

  • Acordar
  • Ligar para o joão
  • Tomar café
  • Escovar os dentes
